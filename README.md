## Proyectos destacados

### BitCafe — *Sistema POS Multi-capa*
**FastAPI · PyQt6 · Jetpack Compose · SQLAlchemy · Mercado Pago**

Arquitectura de tres capas: backend FastAPI con autenticación JWT y pagos Mercado Pago, dos clientes desktop PyQt6 (v2 moderna MVC, v1 legacy), y app Android nativa con Jetpack Compose, Hilt DI, Room persistencia y WorkManager sincronización.

`203 archivos · 15.7 MB · complejidad: alta`

### Orientatec — *Plataforma Web (Producción)*
**PHP · JavaScript · MySQL · Apache**

Plataforma de orientación universitaria desplegada en producción con Apache. Mapa interactivo del campus, autenticación multi-rol (invitado/regular/admin), panel de administración, sistema FAQ y base de datos espacial para POIs georreferenciados.

`178 archivos · 12.5 MB · complejidad: media`

### PracticasUnidas — *Computer Vision Desktop*
**OpenCV · NumPy · SciPy · Matplotlib · Pillow · ttkbootstrap**

Aplicación desktop de procesamiento de imágenes con arquitectura MVC. Más de 30 operaciones: 12 filtros (mediana adaptativa, contraarmónico, bilateral), 6 detectores de bordes (Sobel, Canny, Laplaciano, Kirsch), 5 métodos de segmentación (Otsu, entropía Kapur, umbral dual/banda), ecualización de histograma, morfología y transformaciones de color.

`22 archivos · 6.1 MB · complejidad: alta`

### Union Zalor — *Comunidad Web Gaming*
**PHP · JavaScript · MySQL**

Plataforma web gaming multi-página con autenticación de usuarios, secciones de música, juegos, horarios, reglas y sistema de soporte FAQ.

`188 archivos · complejidad: media`
